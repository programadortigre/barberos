📘 Guía Rápida: MySQL en Docker (modo consola)

🚀 Entrar al contenedor MySQL

Desde tu proyecto con Docker Compose:

docker compose exec mysql mysql -u root -p

Ingresa tu contraseña (por defecto es userpassword si usas el .env que configuramos).

🗂️ Ver bases de datos

SHOW DATABASES;

📂 Usar una base de datos

USE barberia;

📋 Ver todas las tablas

SHOW TABLES;

🔍 Ver estructura de una tabla

Esto te muestra los campos (columnas) y sus tipos:

DESCRIBE nombre_tabla;

Ejemplo:

DESCRIBE roles;

🧾 Ver datos de una tabla

SELECT * FROM nombre_tabla;

Ejemplo:

SELECT * FROM roles;

🧠 Otros comandos útiles

  • SHOW COLUMNS FROM nombre_tabla; → Similar a DESCRIBE
  • EXIT; o quit; → Salir de MySQL
  • \c → Cancelar comando actual
